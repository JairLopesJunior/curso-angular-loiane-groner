// Angular CLI: Criando components, services: ng generate

// Criando componentes ou serviços

> cd nomeProjeto

> ng generate component nomeComponent
               ou
> ng g c nomeComponent

Obs: Por padrão ao criar o service, precisa passar o caminho da pasta onde quer criar.

Ex: ng g s diretiva-ngif/diretiva-ngif // Nome da pasta diretiva-ngif

// Podemos criar os seguintes tipo de arquivos:

- Component
- Service
- Directive
- Pipe
- Class
- Interface
- Enum
