// Angular CLI: Usando pré-processadores (Sass, Less e Stylus)

// Existem duas maneiras que podemos configurar em nosso projeto para utilizar algum dos pré-processadores

// 1- Ao Gerar um novo projeto

> ng new nomeProjeto --style=sass

> ng new nomeProjeto --style=less

> ng new nomeProjeto --style=stylus

// 2- Em um projeto existente

> ng set defaults.styleExt scss

> ng set defaults.styleExt less

> ng set defaults.styleExt styl
